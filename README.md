# Overflood Request

#### Requirements

```javascript
npm install figlet path fs readline cloudscraper
```

#### Installation

```javascript
npm install overflood && mv node_modules/overflood ./ && cd overflood
```

#### Proxy or nonProxy

- can use with proxy 
```javascript
node withproxy.js
```

- or non proxy
```javascript
node nonproxy.js
```
